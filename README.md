# DATN

# How to Install

Step 1: Clone repo
```
$ git clone https://github.com/phamvanbk11/DATN.git

```

Step 2: Install gem dependencies
```
$ bundle install
```

Step 3: Install packages via Bower. We need to install bower before. See http://bower.io/ to install bower if not yet.
```
$ bower install
```

Step 4: Run App
```
$ rails s
```
